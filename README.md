## Live Demo

Hosted on Github Pages [here](https://cinematters.edou.dev/).

## Project Setup

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm run build

# Preview production build
pnpm run preview
```
